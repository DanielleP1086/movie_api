Documentation

This application will provide an array of information (pun intended) about movies which will include: titles with descriptions, genres, and directors. This application uses Express modules and RESTful Api architecture with the the Node.js environment. In addition the app uses the no-sql database, Mongodb, and uses Mongoose for the business logic and data consistency.


Business Logic URL HTTP Method Request Format Response Format
Get list of all movies /movies GET none JSON object
[ { Title: {type: String, required: true}, Description: {type: String, required: true}, Genre: { Name: String, Description: String }, Director: { Name: String, Bio: String }, Actors: [String], ImagePath: String, Featured: Boolean },
]
Get data about single movie /movies/:Title GET none JSON object
{ Title: {type: String, required: true}, Description: {type: String, required: true}, Genre: { Name: String, Description: String }, Director: { Name: String, Bio: String }, Actors: [String], ImagePath: String, Featured: Boolean }
Get data about genres /movies/genres/:Name GET none JSON object(description)
Genre: { Name: String, Description: String }
Get data about directors /movies/directors/:Name GET none JSON object
{ Director: { Name: String, Bio: String }
Create new user profile /users POST JSON object
{ Username: {type: String, required: true}, Password: {type: String, required: true}, Email: {type: String, required: true}, Birthday: Date, FavoriteMovies: [{ type: mongoose.Schema.Types.ObjectId, ref: 'Movie'}] } 		text("new user added")
Update user profile /users/:Username PUT { Username: {type: String, required: true}, Password: {type: String, required: true}, Email: {type: String, required: true}, Birthday: Date, } text("profile updated")
Add favorite movies (list) /users/users/:Username/Movies/:MovieID PUT none text("movie added")
Remove favorite movies (list) /users/:Username/Movies/:MovieID DELETE none text("successfully deleted")
Remove user profile /users/:Username DELETE none text("profile deleted")
