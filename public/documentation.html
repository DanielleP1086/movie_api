Documentation

This application will provide an array of information (pun intended) about movies which will include: titles, genres, and directors. This application uses Express modules and RESTful Api architecture. I will update this page as I add more code.


Business Logic URL HTTP Method Request Format Response Format
Get list of all movies /movies GET none JSON object
[ { title: 'Titanic', director: 'James Cameron', genre: 'disaster' },
{ title: 'The Notebook', director: 'Nick Cassavetes', genre: 'romance' }, ]
Get data about single movie /movies/[title] GET none JSON object
{ title: 'Titanic', director: 'James Cameron', genre: 'disaster' }
Get data about genres /genres GET none JSON object(description)
{ name: 'Romance', description: 'The romance genre focuses on...' }
Get data about directors /directors GET none JSON object
{ name: 'Robert Zemeckis', movies: 'Forest Gump, Cast Away, Flight...' }
Create new user profile /users POST JSON object
{ username:
email adress:
password:
favorites:
} 		text("new user added")
Update user profile /users/[username] PUT none text("profile updated")
Add favorite movies (list) /users/[username]/[favorites] PUT none text("movie added")
Remove favorite movies (list) /users/[username]/[favorites] DELETE none text("successfully deleted")
Remove user profile /users/[username] DELETE none text("profile deleted")
